## How to start
1. Make sure your terminal is opened into this directory:
```
$ cd server/
```

2. Create a python venv as 'venv':
```
$ python3 -m venv venv
```

3. Activate venv:
```
$ source venv/bin/activate
```

4. Install prerequisites:
```
$ python3 -m pip install -r requirements.txt
```

5. Run the project:
```
$ python3 manage.py runserver
```
